# Recruiter Brief

## Best-Fit Roles

cloud SOC analyst, detection engineer, cybersecurity instructor

## Skill Signal

This repository shows practical ability in writing cloud detection logic against safe synthetic logs.

## Why It Matters

Security teams need people who can make judgment visible: define the risk, explain the evidence, document the process, and give other analysts or learners a repeatable path.

## Proof Points

- Working local CLI.
- Synthetic examples under `examples/`.
- Unit tests under `tests/`.
- Roadmap and architecture notes under `docs/`.
- Clear security policy and contribution guardrails.

## Demo Command

```bash
python -m cloudtrail_detection_lab examples/cloudtrail.json
```
